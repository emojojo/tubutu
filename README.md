## 🌟 项目亮点

* **🥬 100% 全满图鉴**：收录了高达 33 种常见有机蔬菜和瓜果的种植日历与栽培要点。
* **🐞 极度硬核的微距病害图库**：拥有 39 张独一无二的高清微距病虫害照片（例如：极度写实的萝卜黄曲条跳甲、蚕豆赤斑病、番茄脐腐病等），帮您快速精准对症下药！
* **🌍 定制化种植日历**：支持切换华北、华南、华东、西南、西北五大气候区，自动适配当地播种时间。
* **💩 环保肥料工坊**：提供环保酵素、有氧堆肥等 7 种家庭自制有机肥的详尽图文教程，实现生态循环。
